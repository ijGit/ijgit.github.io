n 모듈

  • n Node.js 버전을 관리해주는 플러그인
  • 다양한 버전을 선택하여 설치 가능 (최신 버전, 안정 버전, LTS, 특정 버전)

install n

# n모듈 설치
$ npm install -g n

# node 안정버전 설치
$ n stable

# 노드버전 확인
$ node -v
v11.10.0 

# lts버전 설치
$ n lts

# 특정버전 설치 ex> $ n <version> 
$ n 10.0.0

# 버전삭제 
$ n rm 10.0.0
